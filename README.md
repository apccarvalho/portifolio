# 🌐 Andréia Carvalho | Portfolio

Um portfólio pessoal moderno e responsivo, desenvolvido com uma estética **Cyberpunk/Synthwave**. O projeto destaca minha transição de carreira da gestão comercial para **Cloud Computing**, **Infraestrutura** e **Desenvolvimento**.

## 🚀 Tecnologias Utilizadas

* **HTML5 & CSS3**: Estrutura e estilização personalizada com variáveis CSS para o tema dark/neon.
* **JavaScript (Vanilla)**: Consumo dinâmico da **API do GitHub** para listar repositórios públicos em tempo real.
* **Google Fonts**: Utilização das fontes *Syne*, *Space Mono* e *Bebas Neue* para uma tipografia marcante.
* **Intersection Observer API**: Implementação de animações de *fade-in* durante a navegação.

## 📁 Estrutura do Projeto

* **Hero Section**: Apresentação principal com badges de competências, incluindo suporte cloud e programação.
* **Projetos**: Grid automatizado que reflete meus repositórios mais recentes no GitHub.
* **Infra & Labs**: Detalhamento de laboratórios técnicos como Apache, Nginx, DHCP e DNS.
* **Trajetória**: Timeline interativa que organiza minha evolução profissional desde a área de vendas até a análise de suporte técnico.
* **Sobre**: Resumo pessoal que conecta minha disciplina como **Faixa Preta 1º Dan de Taekwondo** com o mundo da tecnologia.
